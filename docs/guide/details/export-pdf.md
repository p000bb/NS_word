# 导出 PDF

将用户上传的 Word 文档（.docx）批量导出为 PDF 格式。

## 功能概述

导出 PDF 功能可自动将选中的多个 DOCX 文件转换为 PDF，支持批量处理、自定义保存目录，以及可选的 PDF 导出选项。处理过程中会实时显示进度。

::: info 实现方式
本功能通过 **pywin32** 调用本机已安装的 **Microsoft Word COM** 实现导出。使用前请确保本机已安装好 **Microsoft Word** 及运行所需的对应软件环境。
:::

## 使用步骤

1. **选择或上传 DOCX**：在应用中选择需要导出为 PDF 的 Word 文档（可多选）。
2. **选择保存位置**：指定 PDF 的输出目录。
3. **（可选）配置 PDF 选项**：根据需要勾选或取消「包含文档属性」「辅助功能文档结构标记」等选项。
4. **开始导出**：确认后执行导出，界面会显示当前处理文件和成功/失败数量。
5. **查看结果**：导出完成后在输出目录中查看生成的 PDF 文件。

::: warning 注意
导出过程中会后台调用本机 Word，请勿在导出时关闭 Word 或占用待导出的 DOCX/目标 PDF 文件。
:::
