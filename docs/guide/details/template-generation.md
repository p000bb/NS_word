# 模板生成

从源 Word 文档生成空白内容的模板文档，保留完整的样式信息。

## 功能概述

模板生成功能可以从源 Word 文档中生成一个标准化的模板文档。生成的模板保留源文档的所有样式和格式，但不包含表格、图片等元素，仅保留文本和样式信息。

::: tip 模板设置
模板是否包含第一页封面内容可以在**模板设置**中进行配置，默认是包含第一页封面内容。
:::

## 作用说明

模板生成的主要作用是为后续的**拆分文档**和**合并文档**功能提供统一的模板 Word 文档，方便统一控制：

- 📐 **样式统一**：确保拆分和合并后的文档使用相同的样式库
- 📄 **页眉页脚**：统一管理文档的页眉页脚格式
- 🔢 **编号系统**：统一控制标题编号、列表编号等编号体系
- 🎨 **格式规范**：保持文档的整体格式风格一致

通过使用统一的模板，可以确保拆分后的各个章节文档和合并后的完整文档在格式上保持完全一致。

## 使用步骤

### 1. 选择源文档

点击"选择文件"按钮，选择要生成模板的源 Word 文档。
![选择源文档](https://raw.githubusercontent.com/p000bb/Image/master/NS_word/2026-01-16-14-17-00.png)

### 2. 选择保存路径

点击"选择文件夹"按钮，选择模板文档的保存位置。
![选择保存路径](https://raw.githubusercontent.com/p000bb/Image/master/NS_word/2026-01-16-14-17-56.png)
::: tip 快速访问
选择保存路径后，可以直接点击显示的文件夹地址，系统会自动打开本地文件夹，方便您查看生成后的模板文件。
:::


### 3. 配置模板设置（可选）

在模板设置中配置是否包含第一页封面内容。
![配置模板参数](https://raw.githubusercontent.com/p000bb/Image/master/NS_word/2026-01-16-14-20-42.png)

::: warning 重要提示
当传入的源文档**没有封面**时，请务必在模板设置中将"模板是否包含第一页封面内容"设置为 **false**，否则生成的模板可能会包含不必要的第一页内容。

**封面识别规则**：系统根据第一个分页符之前的内容来识别封面内容。因此，如果您的文档包含封面，**封面末尾必须包含分页符**，否则系统无法正确识别封面内容。
:::

![封面识别规则](https://raw.githubusercontent.com/p000bb/Image/master/NS_word/20260127181933051.png)

### 4. 生成模板

确认配置无误后，点击"生成模板"按钮。
![生成模板](https://raw.githubusercontent.com/p000bb/Image/master/NS_word/2026-01-16-14-19-21.png)
系统会根据配置提取源文档的内容（默认包含第一页），保留完整的样式信息。生成的模板文件名格式：`源文档名_template.docx`

## 功能特点

- ✅ **灵活配置**：可在模板设置中配置是否包含第一页封面内容，默认包含
- ✅ **样式保留**：完整保留源文档的所有样式和格式，包括样式库、编号体系等
- ✅ **内容精简**：不保留表格、图片等元素，仅保留文本和样式信息
- ✅ **快速生成**：一键生成标准模板，为拆分和合并功能提供统一的格式基础

## 注意事项

::: warning 重要提示
生成的模板 Word 文件**必须**使用 Microsoft Office 打开一次并重新保存替换当前文件，这样做的目的是使用 Office 的自动修复功能，确保文档格式的完整性和兼容性，避免后续生成的拆分或者合并的word文件无法使用office打开。

**封面分页符要求**：如果您的源文档包含封面，请确保封面末尾包含分页符。系统通过识别第一个分页符之前的内容来提取封面，如果封面末尾没有分页符，系统将无法正确识别封面内容。
:::
