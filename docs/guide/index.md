# 功能指南

欢迎使用 NS_word 文档处理工具！本指南将帮助您快速了解和使用本工具的各项功能。

## 功能概述

NS_word 是一款专业的 Word 文档处理工具，旨在解决大型 Word 文档编辑困难、卡顿等问题。通过模板生成、文档拆分、文档合并三大核心功能，帮助您高效管理大型文档项目。

## 核心功能

### 📄 模板生成

从源 Word 文档生成空白内容的模板文档，保留完整的样式信息。模板可以为后续的拆分和合并操作提供统一的格式基础。

**主要特点**：
- 可配置是否包含第一页封面内容
- 完整保留样式库、编号体系、页眉页脚等格式
- 快速生成标准化模板

👉 [查看详细说明](./details/template-generation.md)

### ✂️ 拆分文档

将包含多个章节的大型 Word 文档按一级标题拆分成多个独立的文档，支持智能分组管理和灵活的拖拽操作。

**主要特点**：
- 自动识别一级标题（标题 1 样式）
- 支持配置分组初始拆分章节数（1-15）
- 可自定义跳过标题列表，忽略大小写匹配
- 支持拖拽合并分组、自定义分组名称等灵活操作

👉 [查看详细说明](./details/document-split.md)

### 🔗 合并文档

将多个 Word 文档按照指定顺序合并成一个完整的文档，自动生成目录、表目录、图目录，支持中英文两种语言模式。

**主要特点**：
- 支持拖拽调整文档合并顺序
- 自动生成目录、表目录、图目录
- 支持中文和英文两种目录语言模式
- 实时显示处理状态和进度

👉 [查看详细说明](./details/document-merge.md)

### 📄 导出 PDF

将 Word 文档导出为 PDF 格式，支持高清输出和页面布局调整。

**主要特点**：
- 支持配置PDF导出配置
- 实时显示处理状态和进度

👉 [查看详细说明](./details/export-pdf.md)

### ⚙️ 系统设置

管理系统更新、通知和任务完成后的行为配置，自定义应用程序的使用体验。

**主要配置**：
- 启用自动更新：自动检查并安装系统更新
- 系统通知：接收任务完成等重要通知
- 处理完成后自动打开文件夹：任务完成后自动打开保存文件夹

👉 [查看详细说明](./details/system-setting.md)

## 核心优势

- ⚡ **高效处理**：解决大型 Word 文档编辑卡顿问题，无需启动 Word 应用程序即可处理文档
- 🎯 **智能识别**：自动识别章节标题，支持智能分组和灵活调整
- 📑 **自动生成**：合并后自动生成目录、表目录、图目录，支持中英文两种模式
- 🎨 **样式保留**：完整保留文档格式、样式库、编号体系、页眉页脚等所有格式信息
- 🔧 **灵活配置**：支持多种配置选项，满足不同使用场景的需求
- 📊 **实时反馈**：显示处理状态和进度，失败时提供详细错误信息

## 使用流程

### 典型工作流程

1. **生成模板** → 从源文档生成统一的模板文档
2. **拆分文档** → 使用模板将大型文档拆分成多个小文档
3. **编辑文档** → 分别编辑各个拆分后的文档
4. **合并文档** → 使用模板将编辑后的文档合并成完整文档
