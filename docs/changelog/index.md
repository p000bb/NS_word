## [1.0.0]

### 🎉 初版发布

NS_word 文档处理工具首次发布，包含以下核心功能：

#### 📄 模板生成

- 从源文档提取第一页封面，生成标准模板文档
- 完整保留样式信息（样式库、编号体系、页眉页脚等）
- 为后续文档拆分、文档合并提供统一的格式基础

#### ✂️ 文档拆分

- 智能识别一级标题（标题 1 样式）
- 支持拖拽分组管理，灵活调整拆分策略
- 按章节自动拆分大型文档为多个独立文档

#### 🔗 文档合并

- 支持多文档按顺序合并
- 支持拖拽排序，灵活调整合并顺序
- 自动生成目录、表目录、图目录
- 支持中英文两种目录语言模式

#### ⚙️ 系统设置

- 管理系统更新、通知等配置选项
- 自定义任务完成后的行为


## 未来规划

## 版本说明

版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范：
- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正
