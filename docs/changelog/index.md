## 未来规划

### ✨ 功能增强

- **拆分、合并、模板生成功能扩展**
  - 拆分文档：支持更多自定义拆分规则和分组策略
  - 合并文档：增强合并逻辑，支持更复杂的文档结构
  - 模板生成：提供更多模板配置选项和样式提取方式

- **系统设置功能扩展**
  - 添加更多个性化配置选项
  - 支持导出/导入配置，方便在不同设备间同步设置

### 🎯 高级功能

- **支持自定义配置生成目录、表目录、图目录的域代码**
  - 允许用户自定义目录生成的域代码参数
  - 支持更灵活的目录样式和格式配置
  - 提供域代码编辑界面，方便高级用户自定义

## 版本说明

版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范：
- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正
