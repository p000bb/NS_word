# 更新日志

本文档记录了 NS_word 文档处理工具的所有版本更新内容。

## [1.0.1]

### ✨ 新增功能

- **拆分、合并、模板、系统设置功能**
  - 完善了拆分文档、合并文档、模板生成和系统设置的核心功能
  - 提供了更丰富的配置选项和操作方式

- **自动更新功能**
  - 新增系统自动更新检查功能
  - 支持后台自动检查新版本并提示安装
  - 可在系统设置中配置是否启用自动更新

### 🐛 问题修复

- **修复合并文档时未能够合并 embeddings 关系资源**
  - 修复了合并文档时某些嵌入资源（如图片、对象等）丢失的问题
  - 确保合并后的文档包含所有原始文档的资源


## [1.0.0]

### 🎉 初版发布

NS_word 文档处理工具首次发布，包含以下核心功能：

#### 📄 模板生成

- 从源文档提取第一页封面，生成标准模板文档
- 完整保留样式信息（样式库、编号体系、页眉页脚等）
- 为后续文档拆分、文档合并提供统一的格式基础

#### ✂️ 文档拆分

- 智能识别一级标题（标题 1 样式）
- 支持拖拽分组管理，灵活调整拆分策略
- 按章节自动拆分大型文档为多个独立文档

#### 🔗 文档合并

- 支持多文档按顺序合并
- 支持拖拽排序，灵活调整合并顺序
- 自动生成目录、表目录、图目录
- 支持中英文两种目录语言模式

#### ⚙️ 系统设置

- 管理系统更新、通知等配置选项
- 自定义任务完成后的行为


## 未来规划

### 🔧 问题修复

- **修复生成的文档使用 Word 打开警告的问题**
  - 优化文档生成逻辑，确保生成的文档完全符合 Word 标准格式
  - 消除打开文档时出现的兼容性警告提示

### ✨ 功能增强

- **拆分、合并、模板生成功能扩展**
  - 拆分文档：支持更多自定义拆分规则和分组策略
  - 合并文档：增强合并逻辑，支持更复杂的文档结构
  - 模板生成：提供更多模板配置选项和样式提取方式

- **系统设置功能扩展**
  - 添加更多个性化配置选项
  - 支持导出/导入配置，方便在不同设备间同步设置

### 🎯 高级功能

- **支持自定义配置生成目录、表目录、图目录的域代码**
  - 允许用户自定义目录生成的域代码参数
  - 支持更灵活的目录样式和格式配置
  - 提供域代码编辑界面，方便高级用户自定义

## 版本说明

版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范：
- **主版本号**：不兼容的 API 修改
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正
